# 🤖 Pick and Place Robot Simulation using RoboDK

## 📌 Overview

This project demonstrates a **Pick and Place Robot Simulation** created using RoboDK.
The robot performs automated picking and placing operations in a virtual environment without physical hardware.

RoboDK is a powerful simulation tool used for **robot programming, offline simulation, and industrial automation training**.

---

## 🎯 Objectives

* To understand robotic arm movement in a simulated environment
* To learn industrial robot programming concepts
* To simulate pick and place operations without hardware
* To gain experience with RoboDK software

---

## 🧠 About RoboDK

RoboDK is a simulation software that allows users to:

* Program industrial robots
* Simulate real-world automation tasks
* Test robot movements safely
* Visualize robotic workflows

---

## ⚙️ Working Principle

1. A robot model is selected inside RoboDK
2. Objects are placed in the simulation workspace
3. Target positions are defined (pick & place points)
4. The robot is programmed using RoboDK interface
5. The robot simulates movement to pick and place objects
